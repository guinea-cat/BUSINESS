# SAGE - AI黑客松社会价值评估系统

![SAGE Logo](https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image?prompt=AI%20hacking%20competition%20social%20value%20evaluation%20system%20logo%2C%20modern%20design%2C%20blue%20and%20green%20colors%2C%20professional%20appearance&image_size=square_hd)

## 项目简介

SAGE（Smart AI Hackathon Evaluation System）是一款专为AI黑客松比赛评委设计的社会价值评估工具，通过多维度分析为评委提供专业、客观的社会价值评估报告。

## 核心功能

- **基础项+加分项架构**：伦理安全合规性（20%）+ 核心社会价值亮点（80%）
- **7维度评估框架**：伦理红线检查、隐私与数据保护、算法公平性意识、社会影响深度、环境可持续性、公益普惠导向、长期愿景与变革潜力
- **AI辅助报告**：使用DeepSeek模型生成详细、专业的评估报告
- **评委支持**：提供核心追问问题、潜在风险点和关键考察维度
- **GitHub仓库分析**：直接输入GitHub链接进行项目分析

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **后端**：Python 3.8+
- **AI模型**：DeepSeek
- **Web框架**：Gradio

## 评估体系

### AI黑客松社会价值评估评分表（2:8比例）

#### 一、基础项评估（20分，底线检查）

评估原则：基础项仅检查是否存在问题，符合要求得满分20分。

**伦理安全合规性（20分）**

| 检查维度 | 检查标准 | 发现问题扣分 |
|---------|---------|------------|
| 伦理红线检查 | 是否触及公认伦理红线 | 触及即扣20分 |
| 隐私与数据保护 | 是否有明显隐私风险 | 存在风险扣3-10分 |
| 算法公平性 | 是否有明显不公平设计 | 存在问题扣3-10分 |

**基础项得分** = 20分 - 扣分

**扣分说明**：
- 触及伦理红线（如侵犯人权、恶意设计）：扣20分，社会价值维度直接不及格
- 有明显隐私风险（如不必要数据收集、缺乏保护）：扣3-10分
- 有明显不公平设计（如加剧歧视、忽视弱势群体）：扣3-10分

#### 二、核心亮点项评估（80分）

评估原则：选择项目最突出的1个社会价值维度进行深度评估。

**维度选择表**

| 核心维度 | 适用项目类型 | 评分权重 |
|---------|------------|--------|
| 社会影响深度 | 解决具体社会问题、服务特定群体 | 100% |
| 环境可持续性 | 环保、节能、可持续发展相关 | 100% |
| 公益普惠导向 | 普惠性、可及性、非营利性项目 | 100% |
| 长期愿景与变革潜力 | 有宏大愿景、寻求系统性变革 | 100% |

**各维度评分标准（5分制）**

每个维度下设4个子维度，按5分制评分后加权计算：

**A. 社会影响深度**
- 问题针对性（40%）：5分=精准识别重要问题，1分=问题模糊
- 解决方案适配度（30%）：5分=高度创新匹配，1分=解决方案脱节
- 受益群体聚焦（20%）：5分=明确服务弱势群体，1分=未定义受益群体
- 影响可持续性（10%）：5分=长期可持续设计，1分=一次性影响

**B. 环境可持续性**
- 直接环境效益（50%）：5分=显著可量化效益，1分=无明显效益
- 绿色设计理念（30%）：5分=全面考虑生命周期，1分=忽视环境影响
- 资源效率（15%）：5分=高度优化，1分=资源浪费
- 意识提升作用（5%）：5分=有效促进绿色行为，1分=无意识提升

**C. 公益普惠导向**
- 普惠性设计（40%）：5分=特别降低使用门槛，1分=排斥特定群体
- 公益优先性（40%）：5分=明确非营利模式，1分=完全商业导向
- 数字包容性（15%）：5分=专门服务数字弱势群体，1分=加剧数字鸿沟
- 利益分配公平性（5%）：5分=价值分配高度公平，1分=分配明显不公

**D. 长期愿景与变革潜力**
- 愿景清晰度与合理性（40%）：5分=清晰宏大且合理，1分=缺乏愿景
- 系统性变革潜力（40%）：5分=可能引发系统性变革，1分=无变革潜力
- 实施路径可行性（15%）：5分=清晰实施路径，1分=无可行路径
- 价值取向先进性（5%）：5分=体现先进价值观，1分=价值取向有严重问题

**核心亮点项得分** = Σ(子维度得分×权重) × 16

#### 三、总分与等级（100分）

**总分** = 基础项得分 + 核心亮点项得分

| 总分 | 等级 | 评价标准 |
|-----|-----|--------|
| 90-100 | 卓越 | 社会价值显著，亮点突出 |
| 80-89 | 优秀 | 社会价值明确，表现良好 |
| 70-79 | 良好 | 有一定社会价值 |
| 60-69 | 合格 | 基本符合要求 |
| <60 | 待改进 | 社会价值不足 |

## 报告格式

系统生成的评估报告包含以下部分：

1. **项目基本信息**：项目名称、评估时间、核心社会价值类型
2. **基础项评估**：伦理红线检查、隐私与数据保护、算法公平性意识
3. **核心社会价值亮点深度分析**：选择最突出的1-2个维度进行深度分析
4. **其他维度简要评估**：对剩余维度的简要评价
5. **综合评估与建议**：总体社会价值评价、优化建议、评委关注点
6. **评估置信度**：信息充分性、评估确定性、需要澄清的问题

## 使用指南

### Gradio界面使用

1. 在浏览器中访问 `http://localhost:7862`
2. 在「GitHub仓库URL」输入框中粘贴要分析的GitHub仓库链接
3. （可选）调整各维度权重配置
4. （可选）上传补充资料链接
5. 点击「开始评估」按钮
6. 等待分析完成（通常需要30秒-2分钟）
7. 查看生成的评估报告
